# PowerToys

PowerToys is the easiest way to add a few quality-of-life improvements without turning the machine into a science project.

Recommended modules for this repo:

- Keyboard Manager for key remaps like Caps Lock to Ctrl
- FancyZones if you like predictable tiling
- PowerRename for bulk file cleanup
- Awake for long-running installs or compiles

This scaffold does not overwrite PowerToys settings yet. The repo should eventually offer a documented import path once the configuration format is stable enough to version cleanly.
